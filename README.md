# 2d-plane-3d-simulation
Plane together with rotation and viewer movement projected on 2D screen, making an illusion of 3D

# Keyboard control
| Key  | Function |
| ------------- | ------------- |
| ← | Rotating plane left |
| →  | Rotating plane right  |
| +(numpad)| Viewer forward movement|
| -(numpad)| Viewer backward movement|
